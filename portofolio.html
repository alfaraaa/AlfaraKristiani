Portofolio

Ini adalah beberapa aplikasi dan program sederhana yang pernah saya buat selama sekolah dan magang.

Aplikasi Notes

Aplikasi Notes adalah aplikasi yang saya buat saat Ujian Praktik kenaikan kelas 12,aplikasi ini simpel dan lengkap dengan CRUD.

Buku Tamu

Buku Tamu adalah projek yang saya buat untuk admin,berisi CRUD dan juga ada fitur filter berdasarkan bulan dan tahun.

Grafik Permohonan Informasi

Grafik permohonan informasi ini saya buat dengan javascript yang diambil dengan Query dari Database SQL.

Trust

Trust adalah aplikasi untuk admin berisi tentang agenda dan kegiatan yang ada di Kantor BPSDMD tempat saya magang.


Login Jeatstrem

Login Jeatstrem ini saya buat untuk mempermudah membuat keamanan aplikasi yang menggunakan admin LTE dan laravel.

Informasi Publik

Informasi publik adalah halaman informasi yang saya buat dengan button untuk menghubungkan halaman baru.

Keuangan Flutter

Aplikasi Keuangan ini saya buat untuk versi mobile,namun masih belum bisa otomatis untuk penjumlahan nominalnya.


Blog

Blog ini saya buat menggunakan bootstrap,css biasa dan menggunakan framework next js dan sanity io untuk mengambil datanya.

Dashbooard

Dashboard keuangan ini saya buat menggunakan css biasa,bootstrap dan admin lte.dan menggunakan framework laravel.

Login & Register

Login dan Register saya buat waktu awal awal belajar pemograman dan masih menggunakan css biasa dan php biasa.
